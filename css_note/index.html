CSS层叠样式表

CSS（Cascading Style Sheets，层叠样式表）是一种用于描述HTML文档外观和格式的样式表语言。它允许开发者控制网页的布局、颜色、字体、间距等视觉效果，从而提升用户体验和网站的美观性。CSS通过选择器将样式应用于HTML元素，实现内容与表现的分离，使得网页设计更加灵活和易于维护。

CSS规则：选择器 { 属性: 值; }
