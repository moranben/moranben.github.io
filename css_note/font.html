CSS-字体

CSS中的字体属性用于控制网页中文本的字体样式和外观。

常用的字体属性

font-family: 指定文本的字体系列，可以是具体的字体名称（如Arial, 'Times New Roman'）或通用字体族（如serif, sans-serif,monospace）。

p { font-family: Arial, sans-serif; }

font-size: 设置文本的字体大小，可以使用绝对单位（如px, pt）或相对单位（如em, rem, %）。

谷歌浏览器默认的文字大小为16px。

h1 { font-size: 2em; }

font-weight: 控制字体的粗细程度，常用值有normal, bold, 以及数值（100到900）。

400 = normal 700 = bold

strong { font-weight: bold; }

font-style: 设置字体的样式，常用值有normal, italic（斜体）, oblique（倾斜体）。

em { font-style: italic; }

line-height: 控制文本行间距，可以使用数字、百分比或长度单位。

p { line-height: 1.5; }

font-variant: 用于设置字体的变体，如small-caps（小型大写字母）。

在实际项目中，font-variant 属性的使用频率较低，主要用于特定的排版需求。

font-variant 属性的值有：
normal: 正常字体
small-caps: 小型大写字母
all-small-caps: 包括大写字母在内都转换为小型大写字母
petite-caps: 特小型大写字母（比小型大写字母更小）
all-petite-caps: 包括大写字母在内都转换为特小型大写字母
unicase: 混合大小写，字体设计者决定如何显示
titling-caps: 专为标题设计的大写字母变体

小写字母转换为小型大写字母,比正常的大写字母小,本就是大写的字母保持不变
h2 { font-variant: small-caps; }

复合书写

font属性可以一次性设置多个字体相关的属性，语法如下：

不能更换顺序，并且各个属性间以空格隔开；必须保留 font-size 和 font-family 属性

font: [font-style] [font-variant] [font-weight] [font-size]/[line-height] [font-family];
