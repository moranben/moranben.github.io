CSS-文本

CSS中的文本属性用于控制网页中文本的样式和外观。

常用的文本属性

color: 设置文本的颜色，可以使用颜色名称、十六进制值或RGB/RGBA值。

p { color: red; }

text-align: 设置文本的对齐方式，常用值有left, right, center, justify。

h1 { text-align: center; }

text-decoration: 设置文本的装饰效果，如overline（上划线）、underline（下划线）、line-through（删除线）等。

a { text-decoration: none; }

text-indent: 设置文本的首行缩进。单位可以是长度单位（如px, em）或百分比。

p { text-indent: 2em; }

line-height: 设置文本行间距。单位可以是长度单位（如px, em）或百分比。

p { line-height: 1.5; }

text-transform: 控制文本的大小写转换，常用值有uppercase, lowercase, capitalize。

h2 { text-transform: uppercase; }

letter-spacing: 设置文本字母之间的间距。

p { letter-spacing: 2px; }

white-space: 控制文本的空白处理方式，常用值有normal, nowrap, pre。

pre { white-space: pre; }

复合书写

可以使用简写属性一次性设置多个文本相关的属性，语法如下：

text: [text-align] [text-decoration] [text-transform] [color];
