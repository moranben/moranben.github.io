<input>

<input> 元素用于为基于 Web 的表单创建交互式控件，以便接受来自用户的数据。

属性

该元素包含全局属性

常用属性示例

文本输入框

<input type="text" name="username" id="username" placeholder="请输入用户名"></input>

密码输入框

<input type="password" name="password" id="password" required ></input>

提交按钮

<input type="submit" value="提交"></input>

单选按钮

<label for="male">男</label>
<input type="radio" id="male" name="gender" value="male" required></input>

复选框

<label for="subscribe">订阅新闻邮件</label>
<input type="checkbox" id="subscribe" name="subscribe" value="yes"></input>

文件上传

<input type="file" name="fileUpload" id="fileUpload"></input>

邮箱输入框

<input type="email" name="email" id="email" required></input>

数字输入框

<input type="number" name="quantity" id="quantity" min="1" max="10"></input>

日期输入框

<input type="date" name="birthday" id="birthday"></input>

颜色选择器

<input type="color" name="favcolor" id="favcolor" value="#ff0000"></input>

范围选择器

<input type="range" name="volume" id="volume" min="0" max="100"></input>

隐藏域

<input type="hidden" name="userid" value="12345"></input>

搜索输入框

<input type="search" name="site-search" id="site-search" placeholder="搜索整个站点" aria-label="搜索整个站点"></input>

电话输入框

<input type="tel" name="usrtel" id="usrtel" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" required></input>

时间输入框

<input type="time" name="usrtime" id="usrtime" required></input>

网址输入框

<input type="url" name="usrurl" id="usrurl" required></input>

图像提交按钮

<input type="image" name="submit" src="submit.png" alt="Submit"></input>

带预定义选项的输入框

<input type="text" id="country" name="country" list="countries"></input>
<datalist id="countries">
<option value="中国">
</datalist>
