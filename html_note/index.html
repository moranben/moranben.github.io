HTML学习

HTML 不是编程语言是标记语言

Web标准的构成:结构(html),表现(css),行为(js)

HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。

元素的内容是开始标签与结束标签之间的内容

<!DOCTYPE html>
<html> <!--HTML文档—-->
    <head>
        <meta charset="UTF-8" /> <!--元数据—-->
        <title>HTML标题</title>
    </head>
    <body>
        <h1>...</h1>
        <p>...</p>
    </body>
</html>
