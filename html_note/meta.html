<meta>

<meta> 元素表示那些不能由其他 HTML 元相关元素表示的元数据信息

属性

该元素包含全局属性

常用

页面内容的简短描述

<meta name="description" content="学习HTML">

控制移动设备上的视口行为

<meta name="viewport" content="width=device-width, initial-scale=1.0">
            
viewport属性内容:
width=device-width:将视口宽度设置为设备屏幕 的宽度
initial-scale=1.0:设置页面的初始缩放比例为1 （不缩放）
user-scalable=no:禁止用户缩放页面（不推荐使 用）
minimum-scale=1.0:设置最小缩放比例
maximum-scale=2.0:设置最大缩放比例

作者名称

 <meta name="author" content="XXX">

了解

提供与页面内容相关的关键字

<meta name="keywords" content="HTML, meta标签">

Open Graph协议meta标签

<meta property="og:title" content="内容抬头>

<meta property="og:description" content="内容的简短描述>

<meta property="og:image" content="分享时显示的图片URL">

<meta property="og:url" content="内容的规范URL">

<meta property="og:type" content="内容类型（website、article、video等）">

<meta property="og:site_name" content="网站名称">

Twitter Cards协议meta标签

<meta property="twitter:card" content="卡片类型（summary、summary_large_image等）">

<meta property="twitter:card" content="卡片标题">

<meta property="twitter:description" content="卡片描述">

<meta property="twitter:image" content="卡片图片URL">

<meta property="twitter:site" content="Twitter用户名">

<meta property="twitter:creator" content="内容创建者的Twitter用户名">
