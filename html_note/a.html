<a>

<a> 元素或称（锚元素）可以通过它的 href 属性创建通向其他网页、文件、电子邮件地址、同一页面内的位置或任何其他 URL 的超链接。

属性

该元素包含全局属性

常用属性示例

打开方式

在新窗口/标签页打开
<a href="链接地址" target="_blank">链接文本</a>
在当前窗口/标签页打开
<a href="链接地址" target="_self">链接文本</a>
在父框架集中打开
<a href="链接地址" target="_parent">链接文本</a>

下载链接

<a href="文件地址" download>下载链接文本</a>

带工具提示的链接

<a href="https://www.example.com" title="点击访问示例网站">示例网站</a>

带样式的链接

<a href="链接地址" style="color: red; text-decoration: none;">链接文本</a>

图片链接

<a href="链接地址"><img src="图片地址" alt="描述文本"></a>

链接JavaScript函数

<a href="javascript:alert('Hello World!');">点击我</a>

电子邮件链接

<a href="mailto:example@example.com">发送电子邮件</a>

电话链接

<a href="tel:+1234567890">拨打电话</a>

锚点链接示例：

定义锚点
<h2 id="section2">第二部分</h2>
链接到锚点
<a href="#section2">跳转到第二部分</a>
从其他页面链接到锚点
<a href="page.html#section2">跳转到另一页的第二部分</a>
