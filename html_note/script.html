<script>

<script> 元素规定了当前文档与某个外部资源的关系
<script> 标签通常放在 <head> 或 <body> 部分。建议将脚本放在 <body> 的末尾，以提高页面加载性能。

属性

该元素包含全局属性

常用

内联JavaScript

<script>console.log('Hello, World!');</script>

外部JavaScript文件

默认情况下,脚本会阻塞HTML解析
<script src="script.js"></script>
脚本异步加载,不阻塞HTML解析
<script src="script.js" async></script>
脚本延迟加载,在文档解析完成后执行
<script src="script.js" defer></script>

了解

使用CDN资源并添加SRI验证

<script src="https://cdn.example.com/script.js" integrity="sha384-..." crossorigin="anonymous"></script>

模块脚本ES6

<script> 
type="module" import { function1, function2 } from './module.js';"
</script>
